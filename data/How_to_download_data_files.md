# How to Download Data Files

Large data files should not be uploaded directly to this GitHub repository. GitHub is mainly used for storing code, documentation, progress reports, and small project files. Large files can make the repository slow to download and difficult to maintain.

For this project, large data files are stored on XJTLU Box.

## Data Download Link

Please download the required data files from the following XJTLU Box link:

https://box.xjtlu.edu.cn/smart-link/d17b715d-c15d-4e4c-9839-f5abe1153a1d/

## Instructions for Students

1. Open the XJTLU Box link above.
2. Download the required data files to your own computer.
3. Store the downloaded files in a local folder that is convenient for your work.
4. Do not commit or push large downloaded data files to this GitHub repository.
5. If your code needs to read a data file, use a local file path and explain the required file location in your own notes or README.

If you generate small result files, summary tables, plots, or documentation from the data, you may upload those files to GitHub if they are relevant to your progress and not too large.
